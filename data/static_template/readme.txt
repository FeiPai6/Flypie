/*
 * 离线案例使用前必读
 * ============================================================================
*/

1.运行：
  1) 双击当前目录下 startup.bat 即可运行本案例
  2) 如果点击 startup.bat无法运行，请检查您电脑的网卡是否有问题 ，或者进入 tour 目录，手动点击 localserver.exe 启动

2.部署项目到服务器：
  1) 拷贝tour目录下的所有文件（localServer.exe可以不拷贝）到您的web服务器的相应目录中
  2) 访问对应目录下的index.html文件即可

3.说明：
  1) 在本地直接打开运行，目前只支持windows系统，无法在苹果电脑上和手机上查看，若有需要，请参考第2步，自行部署到web服务器中
