krpano Tools - Unregistered - Test/Demo mode

Your krpano version has not been registered yet.
Therefore krpano will run in Test/Demo mode.

Limitations in Test/Demo mode:

  • A 'krpano demo version' watermark logo inside the krpano viewer.
  • Small krpano logos in all generated images.
  • No 'private' file encryption.
  • Without maps license - a krpano logo inside the maps plugins.

Except of these limitations, the software itself is fully functional and provides the same features as when registered.
